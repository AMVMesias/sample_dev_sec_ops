# Sample DevSecOps Project

Este es un proyecto de demostración para implementar prácticas de **DevSecOps** utilizando Node.js, GitHub Actions y GitHub Pages.

## 🚀 Características

- **CI/CD:** Pipeline automatizado con GitHub Actions.
- **Seguridad:** Escaneo de dependencias y auditorías de seguridad integradas.
- **Pruebas:** Suite de pruebas unitarias con Jest.
- **Despliegue:** Despliegue automático a GitHub Pages.

## 🛠️ Tecnologías

- **Node.js**
- **Jest** (Testing)
- **GitHub Actions** (CI/CD)
- **GitHub Pages** (Hosting)

## 📋 Requisitos

- Node.js v14 o superior
- npm v6 o superior

## 🔧 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/AMVMesias/sample_dev_sec_ops.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```

## 🧪 Pruebas

Para ejecutar las pruebas:
```bash
npm test
```

## 🛡️ Seguridad

Para ejecutar la auditoría de seguridad:
```bash
npm run audit
```
